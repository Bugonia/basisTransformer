# Venue Strategy

Date:

- 2026-07-09

## Reality Check

The one-month goal is aggressive. The paper can become submission-ready in one
month, but whether it can be submitted to a specific top-conference main track
depends on official CFP deadlines.

Known constraint:

- NeurIPS 2026 main-track submission deadlines have already passed by this date.

Working assumption:

- aim for the next suitable ICLR/ICML/NeurIPS-style main-track cycle, while also
  preparing a workshop/position-track version if a high-quality deadline is open
  sooner.

## Primary Target

  than only a mechanistic interpretability note.

## Backup Targets

Use only after checking official deadlines:

- NeurIPS/ICML workshop on mechanistic interpretability or efficient/robust LLMs;
- COLM-style language modeling venue if timeline matches;
- ICLR workshop if main-track timing is not available.

## Submission Strategy

Version A:

- AAAI main-track paper;
- strong controlled experiments;
- pretrained attribution/intervention result.

Version B:

- workshop paper;
- more conceptual framing;
- controlled experiments plus open-model inventory;
- position future applications.

## What Makes It Top-Conference Level

The paper must not read as a small ablation note. It needs:

- a clean conceptual object: direct write access;
- a mathematical formalism reviewers can reuse;
- experiments designed to isolate a confound;
- a bridge to pretrained open models;
- a practical diagnostic, even if small.
